# 🚀 Customer Churn Prediction System

A Machine Learning-powered web application that predicts whether a customer is likely to churn based on their demographic and service usage data.

---

## 📌 Project Overview

Customer churn is a major challenge for subscription-based businesses. This project aims to identify customers who are at risk of leaving, enabling companies to take proactive retention actions.

This system combines **data analysis, machine learning, and web development** to deliver real-time churn predictions through an interactive UI.

---

## 🎯 Objectives

* Predict customer churn using historical data
* Analyze key factors influencing churn
* Provide probability-based predictions
* Build a user-friendly web interface for real-time predictions

---

## 🔍 Key Insights

* Customers with **month-to-month contracts** have higher churn rates
* **Higher monthly charges** increase churn probability
* Lack of **tech support and online security** leads to higher churn
* Customers with **longer tenure** are more likely to stay
